## 2. Страница /bonuses (история бонусов)

- [x] Отображается заголовок «Бонусы»
- [x] Под заголовком отображаются карточки с историей начислений и списаний бонусов

### Формат карточки бонуса
- [x] В каждой карточке отображается дата и время (пример: `29.01.2026 15:09`)
- [x] На следующей строке отображается текст: *«Использование бонусных баллов при заказе №[номер] [количество]»*
- [x] Или текст: *«Начислено за заказ №[номер] +[количество]»*
- [x] Начисленные баллы отображаются зелёным цветом с знаком `+` перед числом
- [x] Списание баллов отображается красным цветом с знаком `-` перед числом
- [x] Между карточками отступ (padding) 16px

## 5. Окно оставления отзыва

- [x] При клике на кнопку «Оставить отзыв» открывается модальное окно
- [x] В окне отображается текст: *«Пожалуйста, оставьте Ваш отзыв о заказе»*

### Звёзды рейтинга
- [x] Отображаются 5 звёзд серого цвета (`#d4d4d4`) по умолчанию
- [x] При наведении на звезду она и все предыдущие звёзды становятся золотистого цвета (`#ffd700`)
- [x] При уходе курсора со звёзд они возвращаются в исходное состояние (`#d4d4d4`)
- [x] При клике на звезду она и все предыдущие звёзды становятся золотистого цвета (`#ffd700`)

### Поле комментария
- [x] Отображается поле с плейсхолдером «Комментарий»

### Успешная отправка отзыва

## 7. Валидация полей профиля

### Поле «Номер телефона»
- [x] Поле недоступно для редактирования
- [x] При наведении курсор меняется на красный перечеркнутый круг
- [x] При клике на поле ничего не происходит

### Поле «Имя»
- [x] Поле принимает только буквы, цифры, символы `-` и `_`
- [x] Максимальная длина — 16 символов
- [x] При успешном изменении имя на бонусной карте обновляется

### Поле «Email»
- [x] Поле принимает только формат: `x@x.x` (локальная часть, `@`, домен, `.`, зона)
- [x] После символа `@` не допускаются специальные символы (кроме точки `.`)

### Поле «Дата рождения»
- [x] Поле принимает только цифры
- [x] Максимальная длина — 8 цифр
- [x] Год не может быть больше текущего
- [x] Месяц не может быть больше 12
- [x] День не может быть больше 31

### Кнопка «Сохранить»
- [x] При частично заполненном или пустом поле «Дата рождения» появляется сообщение: *«Не удалось сохранить изменение. Неверно указан день рождения»*
- [x] При корректно заполненном поле «Дата рождения» появляется сообщение: *«Вы не можете изменить дату рождения без подтверждения администратора сайта. Свяжитесь с администратором сайта для смены данных.»*
- [x] После обновления страницы дата рождения возвращается к исходному значению
